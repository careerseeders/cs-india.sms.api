# cs-india.sms.api
India SMS Api
C# web API to Send free sms to any mobile in India using C#, VB.NET, ASP.NET

Requirement:

User must have 160by2.com or way2sms.com account(free)
This class library uses screen scraping to send SMS to any mobile
